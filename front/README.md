## 版本
node v18.20.0
npm v10.5.0

安装依赖 `npm install`

调试项目 `npm run dev`

构建 `npm run build`

## 目录结构
src

├─api # 接口

├─assets # 静态资源

├─components # 公用组件

│  ├─goods # 商品

│  ├─header # 头部栏

│  ├─info # 个人信息

│  └─order # 订单

├─constant # 常数

├─img # 图片

├─lib # 辅助模块

├─mock # mock数据

├─page # 页面

│  ├─home # 内容页

│  │  ├─admin # 管理员身份

│  │  ├─customer # 用户身份

│  │  └─merchant # 商家身份

│  └─login # 登录页

├─router # 路由

├─store # 状态管理

└─style # 公用样式
